### Pull the GitHub code

```bash
git clone https://github.com/weav-ai/weav-dev.git
```

### Install requirements

```bash
	pip3 install -r requirements.txt
```

### Add variables to `.env` file

```bash
ENV = "https://<env_name>"
AUTH_TOKEN = "eyJhbGci...."

```

### Usage example

```bash
ENV = "https://subdomain.weav.ai"
AUTH_TOKEN = "eyJhbGciOiJ..."
```
